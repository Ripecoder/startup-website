Integration Code

Copy this script tag and paste it into your website's HTML

<script src="https://chatbotpro.io/chatbot.js?key=your_api_key_here"></script>

📝 Installation Guide

  1. Go to your website's HTML file
  2. Paste this script tag before the closing </body> tag
  3. Save and refresh your website
  4. Your chatbot will appear in the bottom-right corner
